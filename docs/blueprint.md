# **App Name**: Tabula Rasa

## Core Features:

- Blank Canvas: Provides a completely blank HTML page for custom development, setting the stage for subsequent feature additions.
- Modular Integration: Enables the seamless integration of UI components and features as they are defined in future updates.

## Style Guidelines:

- Background color: Almost-white (#F5F5F5) to provide a neutral backdrop for any added elements.
- Font: 'Inter', sans-serif, for a clean and universally compatible base, to be adapted as content is defined.
- Initial Layout: A basic, unopinionated layout that adapts to content as it is added, providing maximum flexibility.
